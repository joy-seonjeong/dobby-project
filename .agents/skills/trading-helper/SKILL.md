# Trading Helper Skill 📈

이 스킬은 주식 자동매매 프로그램 개발에 필요한 주요 전략 패턴, 안전성 점검 목록, 그리고 증권사 API 호출 시의 모범 사례를 담고 있습니다.

## 1. 자동매매 핵심 체크리스트
* **민감 정보 보호**: API Key, SecretKey, 계좌 번호 등은 절대 `.py` 코드에 하드코딩하지 않습니다. `.env` 파일로 불러오도록 코드를 강제합니다.
* **주문 이중 전송 방지**: 동일 주문이 여러 번 실행되지 않도록 주문 실행 후 주문 고유 ID 및 처리 상태를 확인하는 체크 로직을 포함합니다.
* **에러 핸들링**: Rate Limit(초당 API 호출 제한) 초과 시 적절한 sleep(예: 0.2초 이상) 처리를 수행합니다.

## 2. 권장 라이브러리
* `requests`: API 호출용
* `pandas`, `numpy`: 시세 데이터 가공 및 보조지표 계산용
* `python-dotenv`: 환경 변수 로드용

## 3. 대표적인 한국투자증권 KIS API 호출 패턴 예시
* 토큰 발급 및 잔고 조회를 수행할 때 필요한 기본 헤더 및 페이로드 구성을 학습하고 적용합니다.
* 모의투자 환경과 실전투자 환경을 전환할 수 있는 `is_mock` 변수나 플래그를 두도록 구성합니다.
