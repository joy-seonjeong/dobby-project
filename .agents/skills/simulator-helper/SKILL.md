# Simulator Helper Skill 📊

이 스킬은 자산 관리 대시보드 및 투자 시뮬레이터를 개발할 때 복리 계산 공식, 포트폴리오 계산, 프론트엔드 차트 호환 데이터 정제에 대한 베스트 프랙티스를 제공합니다.

## 1. 시뮬레이션 수학 모델링
* **매월 적립식 투자 미래 가치 (FV)**:
  $$FV = PMT \times \frac{(1 + r)^n - 1}{r}$$
  * $PMT$: 매월 적립액
  * $r$: 월 이자율 (연 이자율 / 12)
  * $n$: 총 적립 개월수
* **인플레이션율 반영**: 미래 가치를 현재 가치(Real Value)로 환산할 때 실질 이자율($r_{real} = r_{nominal} - i$)을 적용하는 옵션을 필수 제공합니다.

## 2. 포트폴리오 데이터 가공 규칙
* 대시보드 시각화 도구(Recharts 등)에서 사용하기 좋게 `key-value` 쌍을 가진 Array 객체 구조로 데이터를 정제합니다:
  ```json
  [
    { "name": "주식", "value": 50000000, "color": "#FF8042" },
    { "name": "예적금", "value": 30000000, "color": "#0088FE" }
  ]
  ```

## 3. 차트 컴포넌트 룰
* 차트 렌더링 시 반응형 Container(`ResponsiveContainer`)를 사용해 데스크톱과 모바일 기기 모두에 매끄럽게 표시되도록 합니다.
* 가독성을 위해 숫자는 천 단위 컴마(예: `1,000,000원`)로 포맷팅하여 툴팁과 Y축 라벨에 적용합니다.
